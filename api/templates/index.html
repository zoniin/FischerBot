About Fischer Bot

  • Chess engine trained on Bobby Fischer's games
  • Hybrid ML + alpha-beta search algorithm
  • 1972 World Championship style
  • RESTful JSON API

API Endpoints

GET /api/health
Check API health status
POST /api/new_game
Start a new chess game
POST /api/move
Make a move and get bot's response
GET /api/game_state/:game_id
Get current game state

Quick Start

# Start a new game curl -X POST https://your-url.vercel.app/api/new_game \ -H "Content-Type: application/json" \ -d '{"difficulty": "medium"}' # Make a move curl -X POST https://your-url.vercel.app/api/move \ -H "Content-Type: application/json" \ -d '{"game_id": "...", "move": "e2e4"}'
